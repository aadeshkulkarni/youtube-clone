### Youtube spinoff

App link: [Youtubespinoff](https://youtubespinoff.netlify.app)


### How to setup

1. Clone the repository
2. `cd youtube-clone`
3. `npm install`
4. `npm run start`
   
### Tech stack

1. ReactJS
2. TailwindCSS
3. Redux
4. React-router


### Modules

1. Video Listing
2. Video Detail
3. Debounced search
